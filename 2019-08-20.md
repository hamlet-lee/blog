# Docker 运行单节点 Cassandra
9042是客户端使用的端口
```shell
docker run -d --name mycas3_11_4 -p 9042:9042 -d cassandra:3.11.4
```

客户端连接,因为9042是默认端口，不需要再指明
```shell
./cqlsh ${host_name}
```
