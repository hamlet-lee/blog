# 监控Jenkins Job失败
* 使用 Mailer：https://www.youtube.com/watch?v=FyVIr_MQlbw
* 使用API
 * 列出所有job： /api/json
 ```json
 {
    ...,
    "jobs": [
      {
        "name": "xxx",
        "url": "http://.../job/xxx",
        "color": "blue"
      },
      ...
    ]
 }
 ```
  * 查看job的build等：/job/xxx/api/json?pretty=true
  ```json
  {
   ...,
   "builds": [
     {
      "number": 15,
      "url": "http://.../job/xxx/15"
     }
   ]
  }
  ```
 
 * 查看build的情况： /job/xxx/15/api/json?pretty=true
 ```json
  {
   ...,
   "actions": [
     ...
   ],
   "building": false,
   "duration": 3881,
   "estimatedDuration" : 4768,
   "result" : "SUCCESS",
   "timestamp" : 1559287551168
  }
  ```
