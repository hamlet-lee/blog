# Spring MVC 的 URL 映射的问题
```
@RestController("/aaa")
class Xxx {
@RequestMapping(value = "/update", method = {RequestMethod.GET, RequestMethod.POST})
public void update1() {

}

@RequestMapping(value = "update", method = {RequestMethod.GET, RequestMethod.POST})
public void update2() {

}
```
两个RequestMapping的效果是不同的，前者映射到 “/update” 后者映射到 “/aaa/update”
