# age of PID
参考： https://unix.stackexchange.com/questions/238641/get-process-age-from-command-line
```shell
# 命令
ps -o etime= -p "91395"

# 输出
19-18:36:29
```

判断下启动时间是否超过1天
```shell
A=$(ps -o etime= -p "$PID")
D=${A:0:2}
H=${A:3:2}

if [ "$D" == "  " ]; then
  echo "age less than a day!"
fi
```
