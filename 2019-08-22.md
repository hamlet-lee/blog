# MySql 批量插入
参考：https://segmentfault.com/a/1190000008890065
* 单事务
  ```sql
  START TRANSACTION;
  insert ...
  COMMIT;
  ```
* 单insert，多数据
  ```sql
  INSERT INTO `insert_table` (`datetime`, `uid`, `content`, `type`) 
    VALUES ('0', 'userid_0', 'content_0', 0), ('1', 'userid_1', 'content_1', 1);
  ```
