# Logstash 如果字段属于某个集合 drop 之
参考： https://discuss.elastic.co/t/drop-if-message-contains-text-from-a-list-of-strings/142102

简单hardcode

```
if "some text " in [message] or "some text" in [message] {
drop {}
}
```

从文件读取list

```
input {
  generator {
    lines => [
      '222101333',
      '123456789',
      'abc103def',
      'xyz301mno'
    ]
    count => 1
  }
}

filter {
  translate {
    field       => "[message]"
    destination => "[matched]"
    dictionary  => [ "100", "drop",
                     "101", "drop",
                     "102", "drop",
                     "103", "drop" ]
    exact       => true
    regex       => true
  }
  if [matched] == "drop" {
    drop {}
  }
}

output {
  stdout {
    codec => rubydebug {metadata => true}
  }
}
```
